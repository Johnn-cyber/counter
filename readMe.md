# Counter App

A simple, elegant web-based counter application with customizable step increments and visual feedback.

## Features

- **Increment/Decrement**: Use the **+** and **-** buttons to adjust the counter value
- **Adjustable Steps**: Change the increment/decrement step size with options: 1, 5, 10, or 100
- **Reset Button**: Reset the counter to 0 and step value back to 1
- **Visual Feedback**: 
  - Green dots appear when incrementing
  - Orange dots appear when decrementing
- **Modern UI**: Glassmorphism design with gradient background and smooth animations

## Files

- `index.html` - HTML structure and markup
- `script.js` - JavaScript functionality for counter logic and event handling
- `styles.css` - Styling with gradient background, animations, and responsive design

## How to Use

1. Open `index.html` in your web browser
2. Click the **+** button to increase the counter or **-** to decrease it
3. Change the **Step** dropdown to modify how much the counter increases/decreases
4. Click **Reset** to return the counter to 0

## Technologies Used

- HTML5
- CSS3 (Glassmorphism effects, flexbox)
- Vanilla JavaScript
- Google Fonts (Space Mono)

## Installation

No installation required! Simply open `index.html` in any modern web browser.

## Browser Compatibility

Works on all modern browsers that support CSS backdrop-filter and ES6 JavaScript.
